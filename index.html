easyscrollto.js - js library

Github

A tiny JS library that I developed to quickly implement smooth scrolling to page sections driven by page links and buttons.


The basics: This library links buttons with page sections via ID names. Once buttons or hyperlinks and their associated page sections have been named properly, the script facilitates smooth scrolling to each section from anywhere in the page, triggered by button click. There's an adjustable topNavBarHeight value to help you ensure that any persistent navigation bars don't obscure page content.

This is designed to be plug-and-play. Integration into your project is as simple as adding a few ID names to your HTML.

More detail: For more details about implementation, see the Github repository [link] for this project.
